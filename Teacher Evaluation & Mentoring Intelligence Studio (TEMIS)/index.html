📱 APK Conversion Guide

Option A — PWA Install (Simplest)

Open this app in Chrome on Android → tap ⋮ menu → "Add to Home Screen". The app installs like a native app with full offline capability.

Option B — Android Studio WebView Wrapper

1. Open Android Studio → New Project → Empty Activity
2. Replace layout with WebView loading file:///android_asset/index.html
3. Copy index.html, manifest.json, service-worker.js, author.jpg to app/src/main/assets/
4. Enable JavaScript in WebViewClient: webView.settings.javaScriptEnabled = true
5. Enable DOM storage: webView.settings.domStorageEnabled = true (required for LocalStorage)
6. Build → Generate Signed APK → Create keystore → Build Release APK

Option C — PWABuilder.com

Host the app on GitHub Pages or Netlify, then visit PWABuilder.com with the URL to generate a signed APK in minutes.
